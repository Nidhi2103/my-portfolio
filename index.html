PROJECTS

MAIZE DISEASE PREDING USING RESNET AND NLP

This project develops a Maize Crop Disease Prediction System using deep learning to help farmers detect diseases early and improve agricultural productivity. It identifies common maize diseases like Grey Leaf Spot, Northern Corn Leaf Blight, and Common Rust using CNN (ResNet-50) for image-based detection and NLP (LSTM) for text-based symptom analysis. The system allows users to input either images or text for accurate disease prediction, promoting sustainable agriculture and food security.

SIMPLE TO DO LIST USING JAVA

This To-Do List application is built using Java Swing, allowing users to add, remove, complete, and save tasks efficiently. It features a simple UI with a task list, input field, and buttons for easy task management. Tasks are saved in a file (`tasks.txt`) and automatically loaded on startup, ensuring continuity. The project showcases GUI development and file handling in Java.
